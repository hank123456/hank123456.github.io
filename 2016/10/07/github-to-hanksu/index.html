github page搭建的博客，怎么绑定自己的域名

怎么用hexo建站，网上有大把的教程，我也是按照教程弄出来，大家可以自行搜索。

终于建站好了发现域名好丑（太长），于是决定买个域名绑定。最终选择的是万网的，比较了几个国内外的域名商，比如godaddy，发现国内的价格更低，操作更方便。

绑定方法如下：

  • 1、在source文件夹（不是主题的source）中新建一个CNAME文件（无后缀名），然后用文本编辑器打开，在首行添加你的网站域名，如www.hanksu.com，特别注意前面没有http:// ，www有没有好像都行，我是有的，这个可以自行测试，然后使用hexo g && hexo d上传部署。然后你会发现在你的github repository里面根目录下面会多一个CNAME文件。

  • 2、进入万网后台，域名管理下面，点击对应域名的解析按钮。
    a）先添加一个CNAME，主机记录写@，后面记录值写上你的
    xxxx.github.io
    b）再添加一个CNAME，主机记录写www，后面记录值也是xxxx.github.io
    这样别人用www和不用www都能访问你的网站（其实www的方式，会先解析成http://xxxx.github.io ，然后根据CNAME再变成http://xxx.com ， 即中间是经过一次转换的）。

  • 3、等几分钟左，就可以用你的域名访问了。

PS：以上，我们用的是CNAME，也有人使用A记录，后面的记录值是写github page里面的ip地址，不过有时候IP地址会变，导致解析不正确，所以还是推荐用CNAME别名记录要好些，不建议用IP。

PPS：用CNAME的时候，如果遇到你的域名也有邮箱的域名记录，可能CNAME会和MX记录冲突，这个暂时无解，只能二选一了。如果一定要用邮件，那么还是用A记录绑定吧。
